# 人生コックピット

残り時間から逆算して、夢リスト、未来年表、今日の一手を整理する静的Webアプリです。

## GitHub Pagesで公開する

1. GitHubで新しいリポジトリを作る
2. このフォルダ内の `index.html`、`styles.css`、`app.js`、`.nojekyll`、`README.md` をリポジトリ直下に置く
3. `main` ブランチへコミットする
4. Repository Settings → Pages を開く
5. Source を `Deploy from a branch` にする
6. Branch を `main`、folder を `/(root)` にして保存する

公開URLは通常 `https://<GitHubユーザー名>.github.io/<リポジトリ名>/` になります。

設定や夢リストは各ユーザーのブラウザ内に保存されます。サーバーには送信されません。
